About


Pictures Localizer is a mobile application that shows the user his last uploaded pictures on Facebook and the geolocalized ones on Google Maps. The technologies used are Node.js with Express for the Server and Angular.js for the client-side.

The connection with the Facebook and Google Maps APIs is made possible thanks to the open source Javascript SDKs ngFacebook by GoDisco and angular-google-maps. The theme is a free Bootstrap theme created by Start Bootstrap.
